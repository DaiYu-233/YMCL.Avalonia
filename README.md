## 介绍

Yu Minecraft Launcher 是一个跨平台 Minecraft 启动器，支持 Windows，MacOS，Linux 等操作系统，兼容 x64、Arm 架构。

Ymcl 支持 Mod 管理, 游戏自定义, 游戏自动安装 (Forge,Fabric, Quilt, OptiFine), 界面自定义等功能。<del>甚至可以激活 Windows !</del>
